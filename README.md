# Abstract and presentation materials for SciPy 2016

This is the abstract and presentation materials for presenting MDAnalysis at [SciPy 2016](http://scipy2016.scipy.org/) in Austin, TX. The presentation will be given by Oliver Beckstein (Arizona State University).
